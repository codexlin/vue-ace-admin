# 安装

## 环境要求

- Vue 3.3+
- TypeScript 4.5+（可选）
- Node.js 16+

## 安装方式

### 使用 pnpm（推荐）

```bash
pnpm add vue-ace-admin-ui ant-design-vue
```

### 使用 npm

```bash
npm install vue-ace-admin-ui ant-design-vue
```

### 使用 yarn

```bash
yarn add vue-ace-admin-ui ant-design-vue
```

## 版本说明

- `vue-ace-admin-ui` - 组件库核心包
- `ant-design-vue` - 基础组件依赖（必需）
- `vue` - Vue 3 框架（必需）

## 兼容性

| Vue | Ant Design Vue | vue-ace-admin-ui |
|-----|----------------|------------------|
| 3.4+ | 4.0+ | 0.1.0+ |

## 开发环境

```bash
# 克隆项目
git clone https://github.com/codexlin/vue-ace-admin.git

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 启动文档站点
pnpm dev:docs
```
